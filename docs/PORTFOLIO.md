## Case study (copy-paste)

**Title:** AYRO — Streetwear e-commerce + client CMS

**One-liner:** Production React storefront with Sanity CMS, Formspree leads, and Paystack checkout (ZAR) for an independent clothing brand.

**What I built:** Responsive shop (cart, checkout, custom orders, contact), hosted Sanity studio for products and legal copy, Paystack on Vercel serverless, Formspree forms, dark mode, deploy-on-publish hook, client handoff docs.

**Stack:** React, TypeScript, Vite, Tailwind CSS, Sanity, Vercel, Paystack, Formspree

**Role:** Design, development, CMS, payments, deployment, client training
